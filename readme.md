Build Instructions
------------------

On Unix, Linux, BSD, and OSX:

    make
    sudo make install

This will install Logramm as lgm.

Documentation
-------------

Documentation for Logramm is online, updated daily:

    http://83.212.101.132/

You may access its unix version by:

    man lgm

Or, to view the library reference:

    man lgmlib
